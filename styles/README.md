# Папка стилей

Эта папка содержит все CSS файлы для Canvas Shapes приложения.

## Структура

- `main.css` - Основные стили, общие для всех HTML файлов
- `extended.css` - Дополнительные стили для расширенного примера с дополнительными кнопками

## Использование

### Основное приложение (index.html)
```html
<link rel="stylesheet" href="./styles/main.css">
```

### Расширенный пример (example-extended-buttons.html)
```html
<link rel="stylesheet" href="./styles/main.css">
<link rel="stylesheet" href="./styles/extended.css">
```

## Преимущества разделения стилей

1. **Повторное использование** - Основные стили можно использовать в нескольких HTML файлах
2. **Поддержка** - Легче поддерживать и обновлять стили
3. **Производительность** - Браузер может кэшировать CSS файлы
4. **Чистота кода** - HTML файлы становятся чище и легче читаются
