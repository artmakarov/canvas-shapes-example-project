Демонстрация гибкой системы кнопок

Этот пример показывает, как легко добавлять новые кнопки без изменения основных классов CanvasUIManager и CanvasApp.

Новые кнопки:

  • Randomize - перемешивает фигуры
  • Reset - сбрасывает все фигуры
  • Info - показывает статистику
Кликните Add для создания фигур. Кликните на фигуру для выбора.
